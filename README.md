# REX Retail Sales Performance Dashboard

A complete retail sales analytics dashboard built to track transactions, revenue, product performance, sales trends, and staff performance across multiple regions.  
This project demonstrates strong skills in data cleaning, KPI development, dashboard design, and business performance analysis using Microsoft Excel.

The *REX Retail Dashboard* provides a clear and interactive overview of sales performance across regions such as Abuja, Enugu, Jos, Ilorin, Ibadan and more.

It helps business managers answer key questions such as:

## ğŸ§© Dataset Description

The dataset includes transaction-level retail data with the following fields:

| Column | Description |
|--------|-------------|
| *TransactionID* | Unique code for each transaction |
| *Date* | Date of purchase |
| *Region* | Geographic sales zone (North, West, South) |
| *Branch* | Specific store location |
| *Product Category* | Clothing, Footwear, Accessories, etc. |
| *Product Name* | Name of the product sold |
| *Units Sold* | Quantity purchased |
| *Unit Price* | Price per unit |
| *Discount* | Percentage discount applied |
| *Total Sales* | Final revenue after discount |
| *Salesperson* | Individual responsible for the sale |
| *Payment Method* | Cash, Transfer, POS, etc. |
